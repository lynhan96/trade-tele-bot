# AI Signal System — Final Implementation Plan

## System Philosophy

```
Bot-signal (current):  1 global config → same params for 200 coins → TCP → tele-bot
AI system (new):       scan top 10 hot coins → AI tunes params per coin →
                       run formula logic → signal queue → handleIncomingSignal()
```

**What AI does**: Detect market regime per coin, pick the best formula, tune its parameters.
**What AI does NOT do**: Decide buy/sell. The formula logic (ported from F1–F8) makes that call.
**New feature**: Signal queue — if a position is already active for a coin, the next AI signal queues and auto-executes when the current one resolves.

## Signal State Machine

```
                      ┌─────────────────┐
  AI generates  ──►   │    GENERATED    │
  signal               └────────┬────────┘
                                │
              ┌─────────────────┼──────────────────┐
              ▼                 ▼                  ▼
        No active         Active signal       Same direction
        signal for        exists (diff       as active signal
        this coin         direction)
              │                 │                  │
              ▼                 ▼                  ▼
          ┌──────┐         ┌────────┐          ┌────────┐
          │ACTIVE│         │ QUEUED │          │SKIPPED │
          └──┬───┘         └───┬────┘          └────────┘
             │                 │
     Send to          Wait for ACTIVE to resolve
  handleIncomingSignal()       │
             │         ┌───────┴────────┐
             │         │  ACTIVE closes  │
             │         └───────┬────────┘
             │                 │
             ▼                 ▼
         ┌───────────┐    ┌─────────────────────────┐
         │ COMPLETED │    │ Execute QUEUED signal    │
         │ (TP / SL) │    │ → becomes new ACTIVE     │
         └───────────┘    └─────────────────────────┘
                                     │
                          If signal TTL expired
                          (> signalTtlHours)
                                     ▼
                               ┌──────────┐
                               │CANCELLED │
                               │(stale)   │
                               └──────────┘
```

**Rules:**
- Max **1 ACTIVE + 1 QUEUED** per coin at any time
- New signal arrives while QUEUED exists → **replace QUEUED** (keep freshest)
- QUEUED signal expires after `signalTtlHours` (default: 4h) → auto-CANCELLED
- Only OPPOSITE direction signals get queued (same direction → SKIPPED)

```typescript
// One document per signal generated by the AI system
interface AiSignalDocument {
  _id: ObjectId;

  // Coin identity
  symbol: string;       // 'BTCUSDT'
  coin: string;         // 'BTC'
  currency: string;     // 'USDT'
  exchange: 'BINANCE' | 'OKX';

  // Signal content
  direction: 'LONG' | 'SHORT';
  entryPrice: number;
  stopLossPrice: number;
  stopLossPercent: number;   // e.g. 1.5

  // Strategy metadata
  strategy: 'TREND_EMA' | 'MEAN_REVERT_RSI' | 'BB_CORRELATION' |
            'STOCH_BB_PATTERN' | 'STOCH_EMA_KDJ' | 'RSI_CROSS' | 'RSI_ZONE';
  regime: 'STRONG_TREND' | 'RANGE_BOUND' | 'VOLATILE' | 'BTC_CORRELATION';
  aiConfidence: number;       // 0–100

  // AI params snapshot (what Haiku tuned for this signal)
  aiParams: object;           // AiTunedParams subset for the chosen strategy

  // Signal lifecycle
  status: 'ACTIVE' | 'QUEUED' | 'COMPLETED' | 'CANCELLED' | 'SKIPPED';
  closeReason?: 'POSITION_CLOSED' | 'TTL_EXPIRED' | 'REPLACED_BY_NEW' | 'MANUAL';

  // Position tracking
  sentToUsers: number;        // count of users signal was broadcast to
  positionOpenedAt?: Date;    // when the first user opened the position
  positionClosedAt?: Date;    // when position resolved (detected by monitor)
  exitPrice?: number;
  pnlPercent?: number;        // rough estimate based on entry vs exit

  // TTL for queue expiry
  expiresAt: Date;            // generatedAt + signalTtlHours

  // Timestamps
  generatedAt: Date;
  executedAt?: Date;          // when sent to handleIncomingSignal
  createdAt: Date;
  updatedAt: Date;
}

// Indexes:
// { symbol: 1, status: 1 }                     → find active/queued per coin
// { status: 1, expiresAt: 1 }                   → TTL cleanup cron
// { strategy: 1, closeReason: 1, createdAt: -1 } → performance analysis
// { createdAt: -1 }                              → recent signals
```

## Redis Keys (Final — Real-time State Only)

| Key | TTL | Content |
|---|---|---|
| `candle-close-price:{COIN}:{interval}` | by interval | Close prices[] (shared with auto-trade) |
| `candle-open-price:{COIN}:{interval}` | by interval | Open prices[] |
| `candle-high-price:{COIN}:{interval}` | by interval | **NEW** High prices[] (for Stoch/KDJ) |
| `candle-low-price:{COIN}:{interval}` | by interval | **NEW** Low prices[] (for Stoch/KDJ) |
| `cache:market:scan` | 5 min | All futures tickers (ticker/24hr) |
| `cache:filter:shortlist` | 5 min | FilteredCoin[] top 10 |
| `cache:ai:params:{symbol}` | 1–4h | AiTunedParams (Haiku output) |
| `cache:ai:regime` | 4h | Global MarketRegime (Sonnet output) |
| `cache:ai:call_count:{YYYY-MM-DD-HH}` | 1h | int (rate limit counter) |
| `cache:ai-signal:active:{symbol}` | 8h | AiSignal._id (active signal ref) |
| `cache:ai-signal:queued:{symbol}` | 4h | AiSignal._id (queued signal ref) |
| `cache:ai-signal:state:{symbol}:{strategy}` | 48h | 2-stage state for F4/F5 |

> **Design choice**: Redis holds real-time signal state (fast lookups). MongoDB holds full signal records (persistence, analytics, queuing logic). When tele-bot restarts, it reads MongoDB to restore Redis signal state.

```typescript
@Injectable()
export class AiSignalService {

  // ── Main signal loop ──────────────────────────────────────────────────────

  @Cron('*/30 * * * * *')  // every 30s
  async runSignalScan() {
    const shortlist = await this.coinFilter.getShortlist();

    await Promise.allSettled(
      shortlist.map(coin => this.processOneCoin(coin))
    );
  }

  private async processOneCoin(coin: FilteredCoin) {
    // 1. Get or refresh AI params
    const params = await this.getOrRefreshParams(coin.symbol);
    if (!params || params.confidence < params.minConfidenceToTrade) return;

    // 2. Run rule engine for selected strategy
    const signal = await this.ruleEngine.evaluate(params, coin);
    if (!signal) return;

    // 3. Route through signal queue (handles state machine)
    await this.signalQueue.handleNewSignal(coin.symbol, signal, params);
  }

  // ── Refresh coin shortlist every 5 min ───────────────────────────────────
  @Cron('0 */5 * * * *')
  async refreshCoinFilter() {
    await this.marketData.fetchTickerScan();
    await this.coinFilter.scanAndFilter();
  }

  // ── Monitor active positions every 30s ────────────────────────────────────
  @Cron(CronExpression.EVERY_30_SECONDS)
  async monitorActiveSignals() {
    await this.positionMonitor.checkAndResolve();
  }

  // ── Clean up expired QUEUED signals every 5 min ──────────────────────────
  @Cron('0 */5 * * * *')
  async cleanupExpiredQueued() {
    const expired = await this.aiSignalModel.find({
      status: 'QUEUED',
      expiresAt: { $lt: new Date() },
    });
    for (const s of expired) {
      await this.signalQueue.cancelSignal(s._id, 'TTL_EXPIRED');
    }
  }
}
```

### `SignalQueueService` — State Machine Core

```typescript
@Injectable()
export class SignalQueueService {

  async handleNewSignal(
    symbol: string,
    signalResult: SignalResult,
    params: AiTunedParams,
  ): Promise<void> {

    const active = await this.getActiveSignal(symbol);

    if (!active) {
      // No active signal → execute immediately
      await this.executeSignal(symbol, signalResult, params, 'ACTIVE');
      return;
    }

    const isSameDirection = (active.direction === 'LONG') === signalResult.isLong;

    if (isSameDirection) {
      // Already signaled this direction — skip
      this.logger.debug(`[Queue] ${symbol}: SKIPPED (same direction as active)`);
      await this.recordSkipped(symbol, signalResult, params);
      return;
    }

    // Opposite direction → replace or set QUEUED
    const existing = await this.getQueuedSignal(symbol);
    if (existing) {
      // Cancel old queued signal, store new one
      await this.cancelSignal(existing._id, 'REPLACED_BY_NEW');
    }

    await this.executeSignal(symbol, signalResult, params, 'QUEUED');
    this.logger.log(
      `[Queue] ${symbol}: ${signalResult.isLong ? 'LONG' : 'SHORT'} QUEUED` +
      ` (waiting for ${active.direction} to close)`,
    );

    // Telegram notification about queued signal
    await this.notifyQueued(symbol, signalResult, active);
  }

  async executeSignal(
    symbol: string,
    signalResult: SignalResult,
    params: AiTunedParams,
    status: 'ACTIVE' | 'QUEUED',
  ): Promise<AiSignalDocument> {

    const signalTtlHours = 4;
    const now = new Date();
    const expiresAt = new Date(now.getTime() + signalTtlHours * 60 * 60 * 1000);

    // Build Mongoose doc
    const doc = await this.aiSignalModel.create({
      symbol,
      coin: symbol.replace('USDT', '').toLowerCase(),
      currency: 'usdt',
      exchange: 'BINANCE',
      direction: signalResult.isLong ? 'LONG' : 'SHORT',
      entryPrice: signalResult.entryPrice,
      stopLossPrice: signalResult.stopLossPrice,
      stopLossPercent: params.stopLossPercent,
      strategy: params.strategy,
      regime: params.regime,
      aiConfidence: signalResult.confidence,
      aiParams: params,
      status,
      expiresAt,
      generatedAt: now,
    });

    // Store ref in Redis for fast lookup
    const redisKey = status === 'ACTIVE'
      ? `cache:ai-signal:active:${symbol}`
      : `cache:ai-signal:queued:${symbol}`;
    await this.redis.set(redisKey, doc._id.toString(), status === 'ACTIVE' ? 8 * 3600 : 4 * 3600);

    if (status === 'ACTIVE') {
      // Broadcast to all users via handleIncomingSignal()
      await this.broadcastSignal(doc);
      await this.aiSignalModel.findByIdAndUpdate(doc._id, {
        executedAt: new Date(),
        sentToUsers: await this.countEligibleUsers(),
      });
    }

    return doc;
  }

  async activateQueuedSignal(symbol: string): Promise<void> {
    const queuedId = await this.redis.get(`cache:ai-signal:queued:${symbol}`);
    if (!queuedId) return;

    const queued = await this.aiSignalModel.findById(queuedId);
    if (!queued) return;

    // Check if still valid (not expired)
    if (queued.expiresAt < new Date()) {
      await this.cancelSignal(queued._id, 'TTL_EXPIRED');
      this.logger.log(`[Queue] ${symbol}: queued signal expired, not executing`);
      return;
    }

    // Execute the queued signal
    this.logger.log(
      `[Queue] ${symbol}: activating queued ${queued.direction} signal`,
    );

    await this.aiSignalModel.findByIdAndUpdate(queued._id, {
      status: 'ACTIVE',
      executedAt: new Date(),
    });

    await this.redis.set(`cache:ai-signal:active:${symbol}`, queuedId, 8 * 3600);
    await this.redis.del(`cache:ai-signal:queued:${symbol}`);

    // Broadcast
    await this.broadcastSignalById(queued._id);
  }

  async resolveActiveSignal(
    symbol: string,
    closeReason: string,
    exitPrice?: number,
  ): Promise<void> {
    const activeId = await this.redis.get(`cache:ai-signal:active:${symbol}`);
    if (!activeId) return;

    const pnlPercent = exitPrice
      ? await this.calculatePnl(activeId, exitPrice)
      : undefined;

    await this.aiSignalModel.findByIdAndUpdate(activeId, {
      status: 'COMPLETED',
      closeReason,
      positionClosedAt: new Date(),
      exitPrice,
      pnlPercent,
    });

    await this.redis.del(`cache:ai-signal:active:${symbol}`);

    // Update coin profile stats
    await this.updateCoinProfile(symbol, activeId);

    // Activate queued signal if one exists
    await this.activateQueuedSignal(symbol);
  }

  private async broadcastSignal(signal: AiSignalDocument): Promise<void> {
    await this.telegramBotService.handleIncomingSignal({
      coin: signal.coin,
      currency: signal.currency,
      equity: signal.direction,
      entry: signal.entryPrice,
      stopLoss: signal.stopLossPrice,
      botType: 'BOT_FUTURE_AI_1',
      tradingPairType: 'FUTURE',
      period: signal.aiParams?.primaryKline || '15m',
      isManual: false,
    });
  }

  async getActiveSignal(symbol: string): Promise<AiSignalDocument | null> {
    const id = await this.redis.get(`cache:ai-signal:active:${symbol}`);
    if (!id) return null;
    return this.aiSignalModel.findById(id);
  }

  async getQueuedSignal(symbol: string): Promise<AiSignalDocument | null> {
    const id = await this.redis.get(`cache:ai-signal:queued:${symbol}`);
    if (!id) return null;
    return this.aiSignalModel.findById(id);
  }
}
```

### `PositionMonitorService` — Detects Position Close

```typescript
@Injectable()
export class PositionMonitorService {

  // Called by AiSignalService cron every 30s
  async checkAndResolve(): Promise<void> {
    // Find all symbols with ACTIVE signals
    const activeSignals = await this.aiSignalModel.find({ status: 'ACTIVE' });
    if (activeSignals.length === 0) return;

    const symbols = activeSignals.map(s => s.symbol);

    // Get current open positions from Binance (using monitor account)
    const openPositions = await this.getOpenPositionSymbols();

    for (const signal of activeSignals) {
      const isStillOpen = openPositions.has(signal.symbol);

      if (!isStillOpen) {
        // Position closed (TP hit, SL hit, or manually closed)
        const currentPrice = await this.getCurrentPrice(signal.symbol);
        const closeReason = this.detectCloseReason(signal, currentPrice);

        this.logger.log(
          `[Monitor] ${signal.symbol}: position CLOSED (${closeReason})` +
          ` entry=${signal.entryPrice} current=${currentPrice}`,
        );

        await this.signalQueue.resolveActiveSignal(
          signal.symbol,
          closeReason,
          currentPrice,
        );
      }
    }
  }

  private async getOpenPositionSymbols(): Promise<Set<string>> {
    // Use the monitor account (BOT_FUTURE_AI_1 user or a dedicated account)
    // This is a lightweight call — single API call returns all open positions
    try {
      const monitorUserId = this.configService.get('AI_MONITOR_TELEGRAM_ID');
      if (!monitorUserId) return new Set(); // graceful degradation

      const userData = await this.redisService.get(`user:${monitorUserId}:data`);
      if (!userData?.apiKey) return new Set();

      const positions = await this.binanceService.getOpenPositions(
        userData.apiKey,
        userData.apiSecret,
      );

      // Only include positions with actual open quantity
      return new Set(
        positions
          .filter(p => Math.abs(parseFloat(p.positionAmt)) > 0)
          .map(p => p.symbol),
      );
    } catch (err) {
      this.logger.warn(`[Monitor] Failed to fetch positions: ${err.message}`);
      return new Set();
    }
  }

  private detectCloseReason(
    signal: AiSignalDocument,
    currentPrice: number,
  ): string {
    if (!currentPrice) return 'POSITION_CLOSED';

    const slHit = signal.direction === 'LONG'
      ? currentPrice <= signal.stopLossPrice
      : currentPrice >= signal.stopLossPrice;

    return slHit ? 'POSITION_CLOSED' : 'POSITION_CLOSED';
    // For now use generic. Can refine with TP/SL detection later.
  }

  private async getCurrentPrice(symbol: string): Promise<number | undefined> {
    const scan = await this.redis.get<any>('cache:market:scan');
    if (!scan) return undefined;
    const ticker = scan.find((t: any) => t.symbol === symbol);
    return ticker ? parseFloat(ticker.lastPrice) : undefined;
  }
}
```

### `AiOptimizerService` — Claude API Calls (Efficient)

```typescript
@Injectable()
export class AiOptimizerService {

  // Called when cache:ai:params:{symbol} is missing or expired
  async tuneParamsForSymbol(symbol: string): Promise<AiTunedParams | null> {
    // Rate limit: hard cap 30 Haiku per hour
    if (!await this.checkRateLimit('haiku', 30)) {
      this.logger.warn('[AI] Haiku rate limit reached, using defaults');
      return this.getDefaultParams(symbol);
    }

    // Pre-compute indicators (no AI needed, pure math)
    const indicators = await this.preComputeIndicators(symbol);
    if (!indicators) return this.getDefaultParams(symbol);

    // Get global regime (from Sonnet cache)
    const globalRegime = await this.redis.get<string>('cache:ai:regime') || 'RANGE_BOUND';

    // Get coin profile (historical performance)
    const profile = await this.coinProfileModel.findOne({ symbol });

    const prompt = this.buildHaikuPrompt(symbol, indicators, globalRegime, profile);

    try {
      const response = await this.anthropic.messages.create({
        model: 'claude-haiku-4-5-20251001',
        max_tokens: 512,
        messages: [{ role: 'user', content: prompt }],
      });

      const params = this.parseHaikuResponse(response.content[0].text);
      if (!params) return this.getDefaultParams(symbol);

      // Store in Redis with dynamic TTL based on regime
      const ttl = params.regime === 'VOLATILE' ? 3600 : 4 * 3600; // volatile = 1h, else 4h
      await this.redis.set(`cache:ai:params:${symbol}`, params, ttl);

      // Log to MongoDB for cost tracking
      await this.regimeHistoryModel.create({
        symbol,
        regime: params.regime,
        confidence: params.confidence,
        modelUsed: 'haiku',
        promptTokens: response.usage.input_tokens,
        outputTokens: response.usage.output_tokens,
        costUsd: (response.usage.input_tokens * 0.00025 + response.usage.output_tokens * 0.00125) / 1000,
        reasoning: params.reasoning || '',
        assessedAt: new Date(),
      });

      await this.incrementRateLimit('haiku');
      return params;

    } catch (err) {
      this.logger.error(`[AI] Haiku call failed for ${symbol}: ${err.message}`);
      return this.getDefaultParams(symbol);
    }
  }

  // Pre-compute regime indicators server-side (NO AI, pure math, ~2ms)
  async preComputeIndicators(symbol: string): Promise<RegimeIndicators | null> {
    const closes = await this.redis.get<number[]>(`candle-close-price:${symbol.replace('USDT','').toUpperCase()}:15m`);
    if (!closes || closes.length < 200) return null;

    const ema20 = this.calcEma(20, closes);
    const ema50 = this.calcEma(50, closes);
    const ema200 = this.calcEma(200, closes);
    const rsi14 = this.calcRsi(14, closes);
    const bb20 = this.calcBB(20, 2, closes);

    const currentPrice = closes[closes.length - 1];

    // EMA alignment
    const emaTrend =
      currentPrice > ema20 && ema20 > ema50 && ema50 > ema200 ? 'BULLISH' :
      currentPrice < ema20 && ema20 < ema50 && ema50 < ema200 ? 'BEARISH' : 'MIXED';

    // BB width as % of middle
    const bbWidthPct = ((bb20.upper - bb20.lower) / bb20.middle) * 100;

    // ATR (10-candle)
    const last10 = closes.slice(-11);
    const atrPct = (Math.max(...last10) - Math.min(...last10)) / currentPrice * 100;

    // Volume from market scan
    const scan = await this.redis.get<any[]>('cache:market:scan');
    const ticker = scan?.find(t => t.symbol === symbol);
    const volumeUsd = ticker ? parseFloat(ticker.quoteVolume) : 0;

    // Last 3 candles
    const opens = await this.redis.get<number[]>(`candle-open-price:${symbol.replace('USDT','').toUpperCase()}:15m`);
    const lastCandles = opens ? [2,1,0].map(i => ({
      isGreen: closes[closes.length-1-i] > opens[opens.length-1-i],
      bodyPct: Math.abs(closes[closes.length-1-i] - opens[opens.length-1-i]) / opens[opens.length-1-i] * 100,
    })) : [];

    return {
      symbol,
      currentPrice,
      emaTrend,
      ema20, ema50, ema200,
      bbWidthPct: parseFloat(bbWidthPct.toFixed(2)),
      rsi14: parseFloat(rsi14.toFixed(1)),
      atrPct: parseFloat(atrPct.toFixed(2)),
      volumeUsd,
      lastCandles,
    };
  }

  private buildHaikuPrompt(
    symbol: string,
    ind: RegimeIndicators,
    globalRegime: string,
    profile: AiCoinProfileDocument | null,
  ): string {
    const bestStrategy = profile?.bestStrategy
      ? `Historical best strategy for this coin: ${profile.bestStrategy} (${profile.strategyStats[profile.bestStrategy]?.wins || 0} wins)`
      : 'No historical data yet';

    return `You are a crypto trading strategy optimizer. Select the best strategy for this coin.

COIN: ${symbol}
PRICE: $${ind.currentPrice}

REGIME INDICATORS (pre-computed, 15m TF):
- EMA trend: ${ind.emaTrend} (EMA20=${ind.ema20.toFixed(2)}, EMA50=${ind.ema50.toFixed(2)}, EMA200=${ind.ema200.toFixed(2)})
- Bollinger Band width: ${ind.bbWidthPct}% (narrow<2% = ranging, wide>4% = volatile)
- RSI(14): ${ind.rsi14}
- 10-candle ATR as % of price: ${ind.atrPct}%
- Volume 24h: $${(ind.volumeUsd / 1e6).toFixed(0)}M
- Last 3 candles: ${ind.lastCandles.map(c => `${c.isGreen ? 'GREEN' : 'RED'}(${c.bodyPct.toFixed(1)}%)`).join(', ')}
- Global market regime (BTC analysis): ${globalRegime}
- ${bestStrategy}

STRATEGIES (pick ONE):
- TREND_EMA: fast EMA crosses slow EMA. Best: emaTrend=BULLISH/BEARISH, bbWidth>3%
- MEAN_REVERT_RSI: RSI extreme near EMA200. Best: emaTrend=MIXED, RSI<30 or >70
- STOCH_BB_PATTERN: 3-candle pattern near BB + stoch cross. Best: bbWidth<2.5%
```

### `RuleEngineService` — Formula Logic (Ported from F1–F8)

```typescript
@Injectable()
export class RuleEngineService {

  async evaluate(params: AiTunedParams, coin: FilteredCoin): Promise<SignalResult | null> {
    const symbol = coin.symbol;
    const coinName = symbol.replace('USDT', '').toLowerCase();
    const currentPrice = coin.lastPrice;

    switch (params.strategy) {
      case 'RSI_CROSS':
        return this.evalRsiCross(params.rsiCross!, coinName, 'usdt', currentPrice);
      case 'RSI_ZONE':
        return this.evalRsiZone(params.rsiZone!, coinName, 'usdt', currentPrice);
      case 'TREND_EMA':
        return this.evalTrendEma(params.trendEma!, coinName, 'usdt', currentPrice);
      case 'MEAN_REVERT_RSI':
        return this.evalMeanRevertRsi(params.meanRevertRsi!, coinName, 'usdt', currentPrice);
      case 'STOCH_BB_PATTERN':
        return this.evalStochBbPattern(params.stochBbPattern!, coinName, 'usdt', symbol);
      case 'STOCH_EMA_KDJ':
        return this.evalStochEmaKdj(params.stochEmaKdj!, coinName, 'usdt', currentPrice, symbol);
      case 'BB_CORRELATION':
        return this.evalBbCorrelation(params.bbCorrelation!, coinName, 'usdt', currentPrice);
      default:
        return null;
    }
  }

  // Direct port of formula-8.processor.ts checkConfig2()
  private async evalRsiCross(p, coin, currency, currentPrice): Promise<SignalResult | null> {
    const rsiData = await this.indicator.getRsiWithCross(
      currentPrice, coin, currency, p.primaryKline, p.rsiPeriod, p.rsiEmaPeriod, 8
    );
    if (!rsiData) return null;

    let isLong: boolean | null = null;
    if (rsiData.crossedAbove) {
      if (p.enableThreshold && rsiData.rsi.last >= p.rsiThreshold) return null;
      isLong = true;
    } else if (rsiData.crossedBelow) {
      if (p.enableThreshold && rsiData.rsi.last <= p.rsiThreshold) return null;
      isLong = false;
    } else return null;

    if (p.enableHtfRsi) {
      const htf = await this.indicator.getRsiWithCross(currentPrice, coin, currency, p.htfKline, p.rsiPeriod, p.rsiEmaPeriod, 8);
      if (!htf) return null;
      const meetsHtf = isLong ? htf.rsi.last > htf.rsiEma.last : htf.rsi.last < htf.rsiEma.last;
      if (!meetsHtf) return null;
    }

    if (p.enableCandleDir) {
      const candle = await this.indicator.getLastCandle(coin, currency, p.candleKline);
      if (!candle) return null;
      const meetsCandleDir = isLong ? currentPrice > candle.open : currentPrice < candle.open;
      if (!meetsCandleDir) return null;
    }

    const sl = isLong
      ? parseFloat(((1 - p.stopLossPercent / 100) * currentPrice).toFixed(8))
      : parseFloat(((1 + p.stopLossPercent / 100) * currentPrice).toFixed(8));

    return { isLong, entryPrice: currentPrice, stopLossPrice: sl, confidence: 75, strategyUsed: 'RSI_CROSS', conditions: { rsi: rsiData.rsi.last, rsiEma: rsiData.rsiEma.last } };
  }

  // Direct port of formula-8.processor.ts checkConfig3()
  private async evalRsiZone(p, coin, currency, currentPrice): Promise<SignalResult | null> {
    const rsi = await this.indicator.getCurrentRsi(currentPrice, coin, currency, p.primaryKline, p.rsiPeriod, true);
    if (rsi === null) return null;

    let isLong: boolean | null = null;
    if (rsi < p.rsiBottom) isLong = true;
    else if (rsi > p.rsiTop) isLong = false;
    else return null;

    if (p.enableInitialCandle) {
      // Use candle close vs open on current candle — need actual candle data
      const candle = await this.indicator.getLastCandle(coin, currency, p.primaryKline);
      if (!candle) return null;
      const meetsDir = isLong ? candle.close > candle.open : candle.close < candle.open;
      if (!meetsDir) return null;
    }

    if (p.enableHtfRsi) {
      const htf = await this.indicator.getRsiWithCross(currentPrice, coin, currency, p.htfKline, p.rsiPeriod, p.rsiEmaPeriod || 9, 8);
      if (!htf) return null;
      const meetsHtf = isLong ? htf.rsi.last > htf.rsiEma.last : htf.rsi.last < htf.rsiEma.last;
      if (!meetsHtf) return null;
    }

    const sl = isLong
      ? parseFloat(((1 - (p.stopLossPercent || 2) / 100) * currentPrice).toFixed(8))
      : parseFloat(((1 + (p.stopLossPercent || 2) / 100) * currentPrice).toFixed(8));

    return { isLong, entryPrice: currentPrice, stopLossPrice: sl, confidence: 70, strategyUsed: 'RSI_ZONE', conditions: { rsi, rsiTop: p.rsiTop, rsiBottom: p.rsiBottom } };
  }

  // F4 — 2-stage stateful: requires OHLC cache
  private async evalStochBbPattern(p, coin, currency, symbol): Promise<SignalResult | null> {
    const stateKey = `cache:ai-signal:state:${symbol}:STOCH_BB`;
    const state = await this.redis.get<StochBbState>(stateKey);
    const ohlc = await this.indicator.getCandleOhlc(coin, currency, p.primaryKline);
    if (!ohlc) return null;

    if (state && state.count > 0) {
      // Stage 2: wait for stoch cross confirmation
      const stoch = this.calcStoch(ohlc, p);
      const bbValue = this.calcBB(ohlc.closes, p.bbPeriod, p.bbStdDev);
      const bbRange = bbValue.upper - bbValue.lower;
      const closePrice = ohlc.closes[ohlc.closes.length - 1];

      const isCrossAbove = this.indicator.crossedAbove(stoch.k, stoch.d);
      const isCrossBelow = this.indicator.crossedBelow(stoch.k, stoch.d);

      let hasSignal = false;
      if (state.isLong && isCrossAbove && stoch.d.last < p.stochLong) {
        const pct = Math.abs(closePrice - bbValue.lower) / bbRange * 100;
        if (closePrice <= bbValue.lower || pct < p.rangeCondition2) hasSignal = true;
      }
      if (!state.isLong && isCrossBelow && stoch.d.last > p.stochShort) {
        const pct = Math.abs(closePrice - bbValue.upper) / bbRange * 100;
        if (closePrice >= bbValue.upper || pct < p.rangeCondition2) hasSignal = true;
      }

      if (state.count >= p.numberCheckCandle) {
        await this.redis.del(stateKey);
        return null;
      }

      await this.redis.set(stateKey, { ...state, count: state.count + 1 }, 48 * 3600);

      if (!hasSignal) return null;

      await this.redis.del(stateKey);
      const sl = state.isLong
        ? parseFloat(((1 - p.stopLossPercent / 100) * closePrice).toFixed(8))
        : parseFloat(((1 + p.stopLossPercent / 100) * closePrice).toFixed(8));

      return { isLong: state.isLong, entryPrice: closePrice, stopLossPrice: sl, confidence: 80, strategyUsed: 'STOCH_BB_PATTERN', conditions: { stochK: stoch.k.last, stochD: stoch.d.last } };

    } else {
      // Stage 1: detect 3-candle reversal pattern near BB
      const closes = ohlc.closes;
      const opens = ohlc.opens;
      const bbValue = this.calcBB(closes, p.bbPeriod, p.bbStdDev);
      const bbRange = bbValue.upper - bbValue.lower;
      const n = closes.length;

      const c0 = { close: closes[n-1], open: opens[n-1] }; // latest
      const c1 = { close: closes[n-2], open: opens[n-2] };
      const c2 = { close: closes[n-3], open: opens[n-3] };

      // RED-GREEN-RED → LONG
      if (c0.close < c0.open && c1.close > c1.open && c2.close < c2.open) {
        const pct = Math.abs(c0.close - bbValue.lower) / bbRange * 100;
        if (c0.close <= bbValue.lower || pct < p.rangeCondition1) {
          await this.redis.set(stateKey, { isLong: true, count: 1 }, 48 * 3600);
        }
      }
      // GREEN-RED-GREEN → SHORT
      if (c0.close > c0.open && c1.close < c1.open && c2.close > c2.open) {
        const pct = Math.abs(c0.close - bbValue.upper) / bbRange * 100;
        if (c0.close >= bbValue.upper || pct < p.rangeCondition1) {
          await this.redis.set(stateKey, { isLong: false, count: 1 }, 48 * 3600);
        }
      }

      return null; // Stage 1 never returns a signal directly
    }
  }
}
```
